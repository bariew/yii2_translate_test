# Docker Install
1. Запустите докер: 
```
docker compose build ; docker compose up
```
2. Инициализируйте проект:
```
docker compose exec frontend sh -c "composer install ;  php init --env=Development --overwrite=n ; php yii migrate --interactive=0"
```

# Задача: Фильтрация переводчиков по их занятости
### 1. В Бэкенде добавить для Переводчиков поле "availability".
### 2. В Фронтенде На странице редактирования Переводчика добавить возможность задавать значения для этого поля: "Рабочие дни", "Выходные", "Недоступен" 
### 3. В API возвращать эти данные при выдаче списка переводчиков и добавить метод выдачи списка возможных значений этого поля
### 4. В API список доступных переводчиков для Проекта должен учитывать их занятость (если заняты на других проектах или отмечены как "Недоступен" в availability)
### 5. В Фронтенде на странице редактирования Проекта выдавать список доступных переводчиков с пометкой у каждого значения его availability
### 6. В Фронтенде на странице списка проектов отображать прикрепленных переводчиков в разных столбцах согласно их availability
